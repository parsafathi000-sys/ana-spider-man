# Spider Panel — music folder
#
# Drop your audio files here and the panel will play a random one every time
# it opens (toggle in Settings -> "Music on open").
#
# Supported formats: .mp3 .ogg .wav .m4a .webm .aac .flac
#
# This folder is served at /musics and listed by GET /api/settings/music/list.
# The .gitkeep file keeps the (otherwise empty) directory in git.
